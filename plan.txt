==== explicit plan ====


2단계
- model_loader.cpp 에서 Assimp가 읽어온 material의 diffuse texture 경로를 계속 SceneRaw::Material.texture_diffuse 에 저장한다.
- 상대 경로 texture는 obj 파일 기준으로 절대/정규 경로로 바꿔서 저장한다.

3단계
- main.cpp 에서 하드코딩된 단일 texture.png 로딩 코드를 제거한다.
- main.cpp 에서 단일 VAO/VBO/EBO 직접 생성 코드를 제거한다.
- main.cpp 는 창 생성, 카메라 업데이트, 프레임 루프, SceneGPUResources.draw 호출만 담당하게 만든다.

4단계
- scene_gpu_resources.cpp 에서 모든 mesh를 GPU에 업로드한다.
- mesh마다 material_index 를 유지한다.
- material마다 diffuse texture를 로딩한다.
- texture가 없거나 로딩 실패 시 1x1 흰색 fallback texture를 사용한다.

5단계
- draw 시 mesh별로 맞는 diffuse texture를 bind 하고 glDrawElements 를 호출한다.
- 최소 목표는 normal map, alpha mask 까지 쓰는 것이 아니라 diffuse texture가 보이게 하는 것이다.

6단계
- clear() 에서 VAO, VBO, EBO, 생성한 texture, shader program 을 모두 해제한다.
- fallback texture는 중복 삭제하지 않게 처리한다.

성공 기준
- OpenGLTest.exe 를 out/build/x64-Debug 에서 실행해도 Sponza obj 를 찾는다.
- Sponza 가 단일 texture 반복이 아니라 material별 texture로 그려진다.
- texture가 일부 없어도 프로그램이 죽지 않는다.
- 종료 시 리소스 정리가 된다.
